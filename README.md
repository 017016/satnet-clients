# OFFICIAL SATNET CLIENTS

## Linux install script:

`curl -fsSL https://raw.githubusercontent.com/017016/satnet-clients/refs/heads/main/linux.sh -o linux.sh && bash linux.sh`

## MacOS/Darwin install script:

`curl -fsSL https://raw.githubusercontent.com/017016/satnet-clients/refs/heads/main/macos.sh -o macos.sh && sudo bash macos.sh`

## Windows install:

Download the .bat file from this GitHub and run it as administrator.
